# Algorithme de classification par règle

Les algorithmes suivant sont pour le moment implémentés :
- APriori
- ID3
- C4.5
- CART

Un autre algorithme hors sujet (ne pouvant pas générer de règles) est aussi présent (voir `NaiveBayes`).
